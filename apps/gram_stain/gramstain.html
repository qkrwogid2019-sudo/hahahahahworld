Gram Stain as Emotional Interface

This project reinterprets Gram staining and classical defense mechanisms as an emotional interaction model.

이 프로젝트는 그람염색과 고전적 방어기제를 감정 인터랙션 모델로 재해석한 실험이다.

Concept

peptidoglycan represents emotion. Retention and loss reflect different emotional coping strategies.

펩티도글리칸은 감정을 의미함 남음과 탈색은 각기 다른 감정 처리 전략을 나타낸다.

Demo

This interaction only works through user engagement.

이 인터랙션은 사용자의 개입이 있을 때만 작동한다.

This only works when someone interacts.
Just like emotions.
