1. 이번 분석의 배경

이전 포스트에서 Northwind 데이터로 매출 둔화 원인을 찾아봤다. 이번엔 다른 질문이다.

"위고비 같은 다이어트 약, 진짜 운동보다 효과적인 거 맞아?"
"헬스장 사업자 입장에서 이 약이 위협인지 기회인지 알고 싶다."

이 질문에 답하려면 데이터가 필요하다. 직접 DB를 만들어서 분석했다.

2. 데이터 준비: 직접 만든 DB

테이블 구조

테이블명 내용 레코드 수
yoyo_rates 요요율 (방법별) 6
muscle_loss 근손실 비율 6
side_effects 부작용 데이터 8
drug_persistence 약물 지속률 7
exercise_bmi 운동량 vs BMI 4
sarcopenia_qol 근감소증 → 삶의 질 5

데이터 출처: STEP 1 Trial, S-LiTE Trial, SURMOUNT-4, JMCP 2024, NHANES 2021-2023

3. 가설 세우기

가설 1: 약물만 쓰면 요요율이 운동보다 높을 것이다.
가설 2: 약물로 감량하면 근손실 비율이 높을 것이다.
가설 3: 운동량과 BMI는 음의 상관관계가 있을 것이다.
가설 4: 근손실 → 삶의 질 저하로 이어질 것이다.

4. SQL로 검증: 가설 1 (요요율)

쿼리

SELECT
    method,
    timeframe,
    weight_regain_percent AS yoyo_rate,
    source
FROM yoyo_rates
WHERE timeframe LIKE '%1년%'
ORDER BY weight_regain_percent DESC;

결과

방법 요요율 출처
Liraglutide 70% S-LiTE Trial
GLP-1 약물만 67% STEP 1 Extension
GLP-1 + 운동 40% S-LiTE Trial
운동만 25% S-LiTE Trial
검증 결과: 가설 1 채택
약물만 쓰면 67-70% 요요, 운동만 하면 25%. 2.7배 차이.

5. SQL로 검증: 가설 2 (근손실)

쿼리

SELECT
    method,
    weight_loss_kg,
    lean_mass_loss_kg,
    lean_mass_loss_percent,
    -- 감량 1kg당 근손실량 계산
    ROUND(lean_mass_loss_kg / weight_loss_kg, 2) AS muscle_per_kg
FROM muscle_loss
ORDER BY lean_mass_loss_percent DESC;

결과

방법 총 감량 근손실 근손실 비율
Semaglutide (STEP 1) 15.3kg 6.92kg 45.2%
Semaglutide (일반) 10kg 3.9kg 39%
GLP-1 일반 10kg 2.6kg 26%
식단만 10kg 2.5kg 25%
식단 + 근력운동 10kg 1.0kg 10%
검증 결과: 가설 2 채택
약물: 26-45% 근손실 vs 식단+운동: 10%. 운동이 근육 보존에 4배 효과적.

6. SQL로 검증: 가설 3 (운동 ↔ BMI)

쿼리: 상관관계 확인

SELECT
    exercise_level,
    sample_size,
    mean_bmi,
    obesity_rate_percent,
    -- 운동 안하는 그룹 대비 차이
    ROUND(31.3 - mean_bmi, 1) AS bmi_diff,
    ROUND(50.1 - obesity_rate_percent, 1) AS obesity_diff
FROM exercise_bmi
ORDER BY mean_bmi DESC;

결과

운동 수준 표본 평균 BMI 비만율 BMI 차이
거의 안함 1,358 31.3 50.1% 기준
가벼운 운동 1,795 30.2 44.4% -1.1
적정 운동 1,393 29.0 35.5% -2.3
고강도 운동 1,689 28.3 32.7% -3.0
검증 결과: 가설 3 채택
운동량 ↑ → BMI ↓ (명확한 음의 상관관계)
고강도 운동 그룹은 비만율이 17.4%p 낮음 (50.1% → 32.7%)

7. SQL로 검증: 가설 4 (근손실 → 삶의 질)

쿼리

SELECT
    metric,
    sarcopenia_group AS with_sarcopenia,
    non_sarcopenia_group AS without,
    difference AS gap
FROM sarcopenia_qol
WHERE difference IS NOT NULL
ORDER BY ABS(difference) DESC;

결과

지표 근감소증 O 근감소증 X 차이
삶의 질 점수 -12.32 기준 -12.32점
우울증 유병률 42.6% 30% +12.6%p
낙상 위험 2배 1배 2배
검증 결과: 가설 4 채택
근감소증 → 삶의 질 12점 하락, 우울증 1.57배, 낙상 2배
약물로 뺀 살의 45%가 근육이면? 이 리스크가 현실이 된다.

8. 추가 분석: 약물 지속률

약이 효과가 있어도, 계속 먹어야 의미가 있다. 현실은?

SELECT
    drug,
    timeframe,
    persistence_rate_percent AS keep_taking,
    100 - persistence_rate_percent AS stopped
FROM drug_persistence
ORDER BY persistence_rate_percent DESC;
약물 기간 지속률 중단률
Semaglutide (Ozempic) 1년 47.1% 52.9%
GLP-1 전체 6개월 46.3% 53.7%
GLP-1 전체 1년 32.3% 67.7%
Wegovy 2년 24% 76%
GLP-1 전체 2년 15% 85%
현실: 2년 후 85%가 약을 끊는다.
끊으면 67% 요요. 결국 돌아온다.

9. 종합: 상관관계 맵

변수 A → 변수 B 관계 강도
약물 의존도 → 요요율 양(+) 강함
약물 사용 → 근손실 양(+) 강함
운동량 → BMI 음(-) 강함
근감소증 → 삶의 질 음(-) 강함
복용 기간 → 중단률 양(+) 중간

10. 결론: 데이터가 말하는 것

위고비는 "위협"이 아니라 "기회"다.

왜?

  • 약물만 쓰면 67% 요요 → 결국 헬스장으로 돌아온다
  • 약물로 뺀 살의 45%가 근육 → 근력운동 필요성 증가
  • 2년 후 85%가 약 중단 → 유지 방법 필요

헬스장 포지션:

"약 쓰지 마세요" (X) → "약 쓰든 안 쓰든, 운동은 보험입니다" (O)

배운 것

  1. 가설 → 쿼리 → 검증 순서가 중요하다
  2. 상관관계를 찾으려면 비교 대상이 있어야 한다
  3. 숫자만 보면 안 된다. "그래서 뭐?"까지 가야 인사이트다
  4. 데이터가 없으면? 직접 DB를 만들면 된다
