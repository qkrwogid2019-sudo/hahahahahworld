Tableau로 추세선과 상관관계를 분석하면서 배운 것들. 특히 "상관관계 ≠ 인과관계"라는 걸 데이터로 직접 확인했다.

분석 방법 선택하기

뭘 보고 싶냐에 따라 분석 방법이 다르다.

목적 방법
관계 보기 산점도 + 추세선, 상관계수(r), R²
인과관계 증명 회귀분석, A/B 테스트, 시계열 분석
패턴/이상치 박스플롯, 히트맵

p-value와 R² 해석하기

p-value (유의확률)

값 의미
p < 0.05 유의미함
p < 0.01 꽤 유의미함
p < 0.0001 매우 유의미함 (우연일 확률 0.01% 미만)

R² (결정계수)

값 설명력
0.3 이하 약함
0.3~0.5 보통
0.5~0.7 괜찮음
0.7 이상 강함
예시: p < 0.0001, R² = 0.513이면?
→ "상관관계는 확실히 있고(p), 변동의 51.3%를 설명함(R²). 나머지 48.7%는 다른 요인."

응용 1: 이익률 vs 판매량

수업 데이터로 개인적으로 분석해봤다.

이익률 추세

발견: 이익률이 시간에 따라 급감하는 음의 상관관계.

판매량 추세

발견: 판매량은 거의 일정함. 추세선이 평평.

인사이트:
판매량은 유지되는데 이익률만 떨어진다 = "팔리는 양은 똑같은데 마진이 줄고 있다"

가능한 원인:
  • 원가 상승 (재료비, 인건비)
  • 할인/프로모션 증가
  • 저마진 상품 비중 증가
  • 경쟁 심화로 가격 인하

응용 2: 부모 학력 vs 자녀 성적

수정: 아웃라이어(이상치)로 인해 상관관계가 성립하지 않음을 확인하여 해당 내용은 삭제하였다.

석사학위 그룹에서 극단치가 많이 발견되어 상관관계 분석 결과가 왜곡되었다. 다행스러운 결과다.

이상치와 가중치

박스플롯으로 이상치를 발견했을 때, 어떻게 처리할지는 목적에 따라 다르다.

상황 처리
이상치 = 노이즈 가중치 낮춤 / 제외
이상치 = 관심 대상 가중치 높임 / 집중 분석
예시:
- 평균 나이 25-26세인데 36세 1명 → 늦깎이 학습자 or 데이터 오류
- 사기 탐지에서 이상 거래 → 이상치가 핵심이므로 가중치 높임

정리

p-value < 0.05 → 유의미함 (관계 있음)
p-value < 0.0001 → 매우 유의미함

R² 해석:
0.3 이하 → 약함
0.3~0.5 → 보통
0.5~0.7 → 괜찮음
0.7 이상 → 강함

상관관계 ≠ 인과관계 (항상 기억!)
