데이터 분석 실수 노트

2026.01.02

오늘 데이터 작업하면서 한 실수들 기록.

1. 중복값 제거 안 함

뭐가 문제였나

같은 사람 중복값을 제외했어야 했는데 안 했음.

→ 피벗테이블에서 값 요약 기준부터 먼저 수정하고 데이터 작업했어야 했음.

2. 인보이스를 합계로 가져옴

뭐가 문제였나

인보이스를 개수로 가져왔어야 내림차순 정렬이 제대로 됐을 것.

합계로 하니까 의미 없는 결과만 나옴.

3. PERCENTRANK.EXC 함수

=PERCENTRANK.EXC(array, x, [significance])
이해한 것

EXC = Exclusive

  • 최솟값 = 0, 최댓값 = 1을 제외(EXC)한다
  • 그래서 항상 0과 1 사이의 값만 나옴
  • 끝값 왜곡 방지용

마지막 파라미터 (significance):

  • 1을 쓰면 → 0.1, 0.2, 0.3... (소수점 첫째 자리)
  • 1을 안 쓰면 → 엑셀 기본값(보통 3)으로 반환 → 0.234 이런 식
  • 대부분 분석/대시보드에서는 1이면 충분함
