오늘 배운 것

MySQL + DBeaver 설치하고 기본 쿼리 연습했다.

환경 세팅

  • Homebrew로 MySQL 설치
  • DBeaver Community 설치
  • CSV 파일 인코딩 변환 (CP949 → UTF-8)

따옴표 국룰

따옴표 용도 예시
` ` 백틱 테이블명, 컬럼명 `상품`
' ' 작은따옴표 문자열 값 '홍길동'

기본 쿼리

-- 데이터베이스 선택
USE 데이터베이스명;

-- 테이블 목록 / 구조
SHOW TABLES;
DESC 테이블명;

-- 조회
SELECT * FROM 테이블명;
SELECT name, age FROM 테이블명;

조건 검색 (WHERE)

-- 같다 / 같지 않다
WHERE country = 'usa'
WHERE name != '홍길동'

-- NULL은 IS NOT 사용
WHERE name IS NOT NULL

LIKE 패턴 검색

-- ~로 시작 / 끝남 / 포함
WHERE name LIKE '홍%'
WHERE email LIKE '%gmail.com'
WHERE name LIKE '%길%'

정렬 / 제한 / 중복제거

ORDER BY name DESC
LIMIT 10
SELECT DISTINCT country FROM ...

범위 / 목록 / 집계

WHERE age BETWEEN 20 AND 30
WHERE country IN ('usa', 'korea')

SELECT COUNT(*) FROM ...
SELECT SUM(price), AVG(price) FROM ...

GROUP BY

-- 그룹별 집계
SELECT country, COUNT(*)
FROM sample21
GROUP BY country

-- 그룹 조건 (HAVING)
HAVING COUNT(*) > 5

삽질 기록

  • SQLite로 연결해놓고 왜 안 되지 함 → MySQL로 다시
  • 한글 깨짐 → CSV 인코딩 UTF-8로 변환
  • 백틱 치는데 ₩ 나옴 → Option + ₩
  • IS NOT 에러 → 값 비교는 !=
  • 따옴표 빼먹음 → 문자열은 무조건 '값'

결론: SQL은 쉬운데 문법이 깐깐하다.
